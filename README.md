# QUICK START GUIDE

## ✅ How to Access The System

### Method 1: Using Browser (EASIEST)
1. Make sure XAMPP is running (Apache = Green)
2. Open your browser
3. Go to: **http://localhost/Taman%20Sinar%20Mahkota%20Residential%20System/**
4. Bookmark this page!

### Method 2: Create Desktop Shortcut
1. Right-click on Desktop → New → Shortcut
2. Paste this URL:
   ```
   http://localhost/Taman%20Sinar%20Mahkota%20Residential%20System/
   ```
3. Name it: "TSM Residential System"
4. Click OK
5. Double-click shortcut to access!

## 🔧 Troubleshooting

**Still seeing PHP code?**
1. Check XAMPP: Apache must be running (green)
2. Verify URL starts with `http://localhost` (not `file://`)
3. Restart Apache in XAMPP

**Can't access?**
- Make sure files are in: `C:\xampp\htdocs\Taman Sinar Mahkota Residential System\`
- Check folder name matches URL (with %20 for spaces)
